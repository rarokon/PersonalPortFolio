About me

Passionate Web Developer

Fast forward to my 12th grade and even into my Bachelor's in Computer Science, web development wasn’t something we specifically studied. But that didn’t stop me. I was already hooked by then, and YouTube became my best friend for learning. I’d spend hours watching tutorials and creating small projects using HTML, CSS, and JavaScript—nothing fancy, but it was enough to keep me going.

As I kept going, I realized it wasn’t enough to know just the front end. I needed to understand how to connect things to the back end, so I got into HTML and started working with Responsive CSS, React and MongoDB. Learning how the front and back ends come together gave me a much clearer picture of how full-scale web applications work.

And, of course, I started using Git to manage my code and collaborate more easily, which was a game-changer. It really prepared me for the kind of teamwork I know I’ll need in the future.
