About me

Passionate Web Developer

Fast forward to my 12th grade and even into my Bachelor's in Computer Science, web development wasn’t something we specifically studied. But that didn’t stop me. I was already hooked by then, and YouTube became my best friend for learning. I’d spend hours watching tutorials and creating small projects using HTML, CSS, and JavaScript—nothing fancy, but it was enough to keep me going.

As I kept going, I realized it wasn’t enough to know just the front end. I needed to understand how to connect things to the back end, so I got into HTML and started working with Responsive CSS, React and MongoDB. Learning how the front and back ends come together gave me a much clearer picture of how full-scale web applications work.

And, of course, I started using Git to manage my code and collaborate more easily, which was a game-changer. It really prepared me for the kind of teamwork I know I’ll need in the future.

What I do

Web Development

Web development, also known as website development, refers to the tasks associated with creating, building, and maintaining websites and web applications that run online on a browser. It may, however, also include web design, web programming, and database management.

Software Development

Software development is the process of creating, testing, maintaining, and designing software applications. It involves using techniques and principles from computer science, engineering, and mathematical analysis. The goal of software development is to create software that is reliable, efficient, and easy to use.

System Development

System development is the process of creating, designing, testing, and implementing a new software application or program. It's a problem-solving process that involves using knowledge to create new knowledge and define a solution.

Security Analysis

Security analysis is the process of evaluating the financial health and potential risks of an investment, such as a stock or bond. The goal is to determine if an asset is fairly valued, underpriced, or overpriced, so that investors can make informed decisions.
